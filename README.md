# Loan Application Analytics Dashboard

## Project Overview

The **Loan Application Analytics Dashboard** is a Data Analytics project developed using **Python, Flask, Pandas, and Plotly**. It analyzes loan application data and presents key business insights through an interactive web dashboard. The dashboard helps users understand loan approval trends, applicant information, and important business metrics using visualizations.

---

## Features

* Analyze loan application data
* Display key business KPIs
* Interactive Plotly charts
* Loan approval distribution analysis
* Property area analysis
* Income vs. Loan Amount visualization
* User-friendly web dashboard

---

## Installation

1. Clone the repository.
2. Install the required libraries:

```bash
pip install flask pandas plotly openpyxl
```

3. Run the application:

```bash
python app.py
```

4. Open your browser and visit:

```text
http://127.0.0.1:5000
```
